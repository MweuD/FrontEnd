All Python programs can call a basic set of functions called built-in ­functions, including the print() , input() , and len() functions you’ve seen before. Python also comes with a set of modules called the standard library. Each module is a Python program that contains a related group of functions that can be embedded in your programs. For example, the math module has mathematics-related functions, the random module has random number–related functions, and so on. Before you can use the functions in a module, you must import the module with an import statement.
